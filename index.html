Uso de GitHub por Primera Vez

Desde la terminal (Comando+Ñ) con GitBash
  1. Escribir: git init
  2. Escribir: git add . (Para que el archivo esté trackeado y se agreguen los cambios)
  3. Escribir: git remote add origin (ingrese link acá del repositorio creado en githu)
  4. Escribir: git branch -m
  5. Escribir: git commit -m "Acá va un comentario/detalle de qué se trata este cambio"
  6. Escribir: git push --set-upstream origin master

Otras indicaciones

  • Para crear una rama: git checkout -b nombrerama
  • Para cambiarme de rama: git checkout nombrerama / git switch nombrerama
  • Si ya se clonó el repositoroio y para actualizar con nuevos archivos/rama: git fetch
  • VSC tiene una interfaz visual para manejar github, se clona el link y en el apartado de source control se pueden agregar los cambios, hacer commits y pushear
